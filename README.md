
# Task Manager App

 A simple task manager app built with MERN stack. It allows users to create, read, update, and delete tasks 

## Features

- Create new task
- View all tasks
- Edit existing task
- Delete task
## Technologies

- MongoDB
- Express.js
- React
- Node.js
